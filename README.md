# ⛏️ Instalación para el funcionamiento

1. Descargar el archivo y colocarlo en algún sitio de tu máquina local
2. Tener Python instalado en la máquina (desde la versión 3.10 o superior), si aún no lo tienes descargado, puedes ingresar a descargarlo desde este enlace a continuación

- [Link de Python](https://www.python.org/)

3. Una ves descargado se recomienda usar un entorno de desarrollo (IDE) que permita la ejecución flexible del programa.
4. Ejecutar en la terminal con la ruta del proyecto en su máquina local `python -m flask --app {nombre_de_la_app} run` para ejecutar el servidor de Flask
5. Finalmente abrir el `index.html` y probar las funcionalidades de la web
